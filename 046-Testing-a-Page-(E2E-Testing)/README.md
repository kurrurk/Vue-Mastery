# Testing a Page (E2E Testing)

🧪 В данном уроке рассмотрено End-to-End (E2E) тестирование приложения. В отличие от модульных и интеграционных тестов, E2E-тесты проверяют приложение целиком, имитируя действия реального пользователя.

## Изученные темы

- Основы End-to-End (E2E) тестирования.
- Проверка полного пользовательского сценария от начала до конца.
- Тестирование взаимодействия нескольких компонентов приложения.
- Автоматическая проверка пользовательских действий и ожидаемого результата.
- Выявление ошибок, возникающих при совместной работе компонентов.

## Практика

- Объединены ранее созданные компоненты в простое приложение для работы с постами.
- Реализованы E2E-тесты для проверки основных пользовательских сценариев.
- Выполнена проверка корректности работы приложения с точки зрения пользователя.
