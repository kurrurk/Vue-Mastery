# Deploying to Netlify

🚀 В данном разделе рассмотрен процесс публикации Nuxt-приложения на платформе Netlify — одном из популярных сервисов для хостинга современных веб-приложений.

## Изученные темы

- Подготовка проекта к публикации.
- Создание и настройка GitHub-репозитория.
- Подключение GitHub-аккаунта к Netlify.
- Импорт существующего проекта через Netlify.
- Автоматическое определение настроек сборки Nuxt-приложения.
- Запуск процесса развёртывания (Deployment).
- Получение публичного URL для доступа к сайту.
- Автоматическое обновление сайта при отправке изменений в GitHub.

## Практика

- Создан удалённый GitHub-репозиторий.
- Выполнена публикация исходного кода проекта.
- Настроена интеграция между GitHub и Netlify.
- Выполнен первый деплой приложения.
- Получен публичный адрес сайта.
- Проверена работа автоматической сборки после внесения изменений в репозиторий.

## Возможности Netlify

Netlify предоставляет:

- Автоматический деплой из GitHub.
- Бесплатный хостинг статических и серверных приложений.
- Управление несколькими версиями сайта.
- Просмотр логов сборки и ошибок.
- Автоматическую публикацию новых версий после каждого Push в репозиторий.
- Простую настройку пользовательских доменов и HTTPS.

## Результат

После завершения настройки проект становится доступен в интернете по уникальному адресу Netlify, а дальнейшее обновление сайта может выполняться автоматически через GitHub без ручной загрузки файлов на сервер.
