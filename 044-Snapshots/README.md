# Snapshot Testing in Vue

📸 В данном уроке были рассмотрены снапшот-тесты во Vue, включая стандартные snapshots и inline snapshots.

## Изученные темы

- Создание и использование стандартных snapshot-тестов.
- Создание и использование inline snapshots.
- Проверка соответствия результата рендеринга компонента ожидаемой разметке.
- Автоматическое обнаружение непредвиденных изменений в пользовательском интерфейсе.
- Различия между стандартными и inline snapshots.

## Примечание

> ⚠️ При запуске тестов в режиме наблюдения (_watch mode_) снапшоты можно обновить, нажав клавишу `u` в консоли. Это полезно в случаях, когда изменения в компоненте являются ожидаемыми и необходимо обновить сохранённый эталонный результат.
