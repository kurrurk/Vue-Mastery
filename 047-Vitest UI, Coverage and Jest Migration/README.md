# Vitest UI, Coverage & Jest Migration

🧪 В заключительном уроке рассмотрены дополнительные возможности Vitest, включая графический интерфейс для работы с тестами, анализ покрытия кода и процесс миграции проектов с Jest на Vitest.

## Изученные темы

- Использование Vitest UI для запуска и анализа тестов.
- Просмотр результатов тестирования через графический интерфейс.
- Настройка и запуск проверки покрытия кода тестами (Coverage).
- Анализ отчётов о покрытии кода.
- Основы миграции тестового окружения с Jest на Vitest.
- Адаптация существующих тестов и конфигурации проекта.

## Практика

- Выполнен запуск тестов через Vitest UI.
- Настроено отображение покрытия кода тестами.
- Рассмотрен процесс переноса тестов из Jest в Vitest.
